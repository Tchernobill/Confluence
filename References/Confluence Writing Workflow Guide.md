## Scene-Specific Workflows

### Writing a Kyr4n POV Scene

**Pre-Writing Focus:**
- [ ] Where is Kyr4n in century-long character arc?
- [ ] Current emotional/psychological state?
- [ ] Recent enhancement costs shown (exhaustion, isolation)?
- [ ] Strategic thinking should be evident
- [ ] Dry wit present even in dark moments

**During Writing:**
- Heavy internal dialogue
- Philosophical undertones when relevant
- Show isolation through internal experience
- Demonstrate protective instinct
- Layer moral complexity

**Post-Writing Check:**
- [ ] Voice feels distinctly Kyr4n (different from arc protagonists)?
- [ ] Internal struggle visible?
- [ ] Strategic thinking demonstrated?
- [ ] Humor balanced with grimness?

---

### Writing an Arc Protagonist POV Scene

**Pre-Writing Focus:**
- [ ] Review this character's distinct voice (Character Bible)
- [ ] What do they misunderstand about Kyr4n?
- [ ] What aspect of Kyr4n do they reveal to reader?
- [ ] Their own character journey this tome?

**During Writing:**
- **CRITICAL**: Voice must sound completely different from Kyr4n
- Show limited perspective on Kyr4n's true nature
- Their interpretation (possibly wrong) of events
- Their own character arc advancing

**Post-Writing Check:**
- [ ] Voice is DISTINCT from Kyr4n and other arc protagonists?
- [ ] Shows Kyr4n from outside perspective?
- [ ] Maintains their limited knowledge?
- [ ] Their character develops, not just serves as Kyr4n observer?

---

### Writing a Multi-Character Scene

**Pre-Writing Focus:**
- [ ] List all characters present
- [ ] Current relationship status between each pair
- [ ] What does each character know/not know?
- [ ] Whose POV? (Only that character's thoughts visible)

**During Writing:**
- Each character's distinct voice in dialogue
- POV character observing/interpreting others
- Information revelation carefully controlled
- Relationship dynamics showing current status

**Post-Writing Check:**
- [ ] Each character sounded like themselves?
- [ ] POV character didn't access others' thoughts?
- [ ] Information revealed appropriately?
- [ ] Relationship dynamics accurate to current state?

---

### Writing Action/Crisis Scenes

**Pre-Writing Focus:**
- [ ] What are character's established capabilities?
- [ ] What are the costs of using those capabilities?
- [ ] What are the stakes based on current relationships?
- [ ] How does this advance plot threads?

**During Writing:**
- Demonstrate capabilities consistent with Character Bible
- **ALWAYS show costs** (enhancement exhaustion, moral weight, etc.)
- Stakes feel real based on relationships
- Pacing appropriate (not too easy, not convenient coincidences)

**Post-Writing Check:**
- [ ] Capabilities consistent with established limits?
- [ ] Costs shown appropriately?
- [ ] No convenient solutions that violate world rules?
- [ ] Update Story Continuity if new capabilities revealed?

**Plot Threads:**
- [ ] All tome-specific threads resolved?
- [ ] Series-spanning threads properly advanced?
- [ ] Nothing accidentally dropped?
- [ ] Clear about what carries to next tome?

**Timeline:**
- [ ] Full tome timeline coherent?
- [ ] Ages tracked correctly throughout?
- [ ] Travel times consistent?
- [ ] Time passage realistic?

**Character Arcs:**
- [ ] Each major character arc advanced appropriately?
- [ ] Emotional progression logical?
- [ ] Relationship changes earned?
- [ ] Physical descriptions maintained?

**World-Building:**
- [ ] No contradictions with established rules?
- [ ] Location details consistent?
- [ ] Technology level maintained?
- [ ] Social structures stable (or changes explained)?

**POV:**
- [ ] Character knowledge tracked correctly?
- [ ] Voice distinction maintained?
- [ ] Perspective consistency within each POV?

#### 2. Create Tome Summary

Write 1-2 page summary covering:
- Major plot developments
- Character arc progression
- Key information revealed
- World-building elements established
- Plot threads active going into next tome
- Foreshadowing planted for future

**Save this summary** - you'll reference it when writing later tomes.

#### 3. Update Master Documents

**Character Bible:**
- Update character arc checkpoints
- Note relationship status at end of tome
- Add any significant personality evolution
- Update age/physical appearance if changed

**Story Continuity Tracker:**
- Mark tome complete in status dashboard
- Verify all tracking current
- Update "Threads Carrying to Next Tome" section
- Flag any unresolved continuity concerns

#### 4. Gap Planning

Before starting next tome:
- [ ] How much time passes between tomes?
- [ ] How do characters change during gap?
- [ ] What happens "off-screen" during gap?
- [ ] How does world evolve during gap?

**Document these answers** before writing next tome.

## Troubleshooting Common Issues

### "I think I contradicted something from Tome 2..."

**Solution Process:**
1. Search your manuscript for the detail in question
2. Check Story Continuity Tracker for established facts
3. Determine which version is correct
4. Update both manuscript and tracker
5. Search for all instances needing change
6. Document fix in "Resolved Continuity Issues" section

**Prevention:** This is why post-writing updates matter!

---

### "This character isn't sounding like themselves anymore..."

**Solution Process:**
1. Open Character Bible to their profile
2. Read their voice description and dialogue samples
3. Review their recent chapters - when did voice drift start?
4. Revise drifted dialogue to match established voice
5. During next session, consciously maintain voice

**Prevention:** Weekly voice-check prevents drift

---

### "I lost track of what everyone knows..."

**Solution Process:**
1. Open Character Knowledge Matrix in Story Continuity Tracker
2. Trace back through timeline when information revealed
3. Update matrix with missing entries
4. Revise scene if character acted on unknown information

**Prevention:** Update knowledge matrix immediately after writing scene with information revelation

---

### "My timeline doesn't make sense anymore..."

**Solution Process:**
1. Create timeline spreadsheet from Story Continuity Tracker
2. Add all chapter events chronologically
3. Calculate time passage between events
4. Find contradictions
5. Determine correct timeline
6. Update manuscript and tracker
7. Verify age calculations

**Prevention:** Update timeline after every chapter, spot-check weekly

### Technique 2: Voice Distinction Test

If unsure whether characters sound distinct:

1. Copy random dialogue from scene
2. Remove character tags
3. Can you identify speaker by voice alone?
4. If no â†’ revise dialogue using Character Bible voice notes

### Technique 3: Continuity Search Strings

Regularly search your manuscript for:
- `[CHECK` - finds all your flagged concerns
- `[UPDATE` - finds notes to update tracking docs
- `[CONTINUITY` - finds flagged continuity issues
- Character names - spot-check descriptions
- Location names - spot-check details
- Time markers - verify timeline

---

### Technique 4: Character Perspective Filter

When writing POV character's observation of another:

**Ask:** 
- How does [POV character] perceive [other character]?
- What do they misunderstand based on limited info?
- What biases affect their interpretation?
- What don't they notice that reader needs to?

Then write observation filtered through that lens.

**Check Character Bible** for relationship status to inform perception.
